## 摘要

本报告对「学术内容诚信检测平台」进行了功能测试与接口测试。测试覆盖用户端（编辑、专家）、管理端（组织管理员、软件管理员）及后端 REST API，重点验证学术检测、检测历史、综合鉴伪报告、人工审核闭环与管理端运维等主路径。

**测试结论**：共设计并执行 **73** 项用例（任务一 43 项、任务二 30 项），**全部通过**，无遗留失败或阻塞项。测试过程中发现 3 项与用例期望不一致的实现问题，已在测评周期内修复并复测通过。性能压测、WebSocket 实时推送、深度跨模态算法等未纳入本轮定量验收。

---

## 1 引言

### 1.1 测试目的

验证系统在需求与概要设计范围内是否满足以下目标：

1. 多角色（编辑、专家、组织管理员）可完成登录与权限隔离；
2. 图像、论文 PDF、Review 文本等检测流程可端到端跑通；
3. 检测历史、综合鉴伪报告、单任务报告可正常查看与下载；
4. 人工审核「申请 → 审批 → 专家鉴定 → 结果汇总」闭环可用；
5. 管理端可完成检测任务监控、资源管理、用户组织查询与系统日志查看；
6. 后端接口对非法输入、越权访问、边界数据具备合理校验。

### 1.2 测试范围


| 类别     | 纳入内容                                                                       |
| ------ | -------------------------------------------------------------------------- |
| 用户端    | 登录/退出、学术检测、检测历史、综合鉴伪报告、人工审核申请、专家审核、个人主页                                    |
| 管理端    | 登录、工作台、检测与资源、用户与组织、人工审核审批、系统日志                                             |
| 后端 API | 认证、上传、检测提交、人工审核、报告下载及边界校验                                                  |
| 不纳入    | 性能指标压测（3s/30s/100 并发等）；WebSocket 依赖 daphne 的实时通知；深度跨模态深度学习融合；精准模式 LLM 效果差异 |

## 5 测试结果说明

### 5.1 主路径验证结论

以下主路径均在测评环境中验证通过：

1. **编辑**：登录 → 学术检测（图像/论文/Review）→ 检测历史 → 综合鉴伪报告 / 单任务报告下载；
2. **人工审核**：编辑发起申请 → 组织管理员通过或拒绝 → 专家鉴定 → 编辑查看汇总；
3. **专家**：登录后仅可见人工审核入口；访问编辑侧路径重定向至专家任务池；
4. **管理端**：检测任务与文件管理、用户组织查询、系统日志（操作留痕 + 检测任务记录）。

## 6 测试期间发现的问题与处理

测评过程中发现以下问题，**均在提交本报告前修复并复测通过**，不计入最终失败项。


| 序号  | 关联用例           | 现象                         | 处理措施                                                 | 复测结果                |
| --- | -------------- | -------------------------- | ---------------------------------------------------- | ------------------- |
| 1   | §6.3-4 / 用例 30 | 管理员拒绝申请未填原因时仍返回 200        | 后端 `handle_reviewRequest` 对 `choice=0` 强制校验 `reason` | 400，提示「拒绝申请时必须填写原因」 |
| 2   | §6.3-2         | 跨用户申请他人检测任务返回 404，未区分「无权」  | 任务存在但非本人 → 403「无权访问该检测任务」；不存在 → 404                  | 403/404 符合设计        |
| 3   | §6.4-5         | Review 文本无长度上限，约 20 万字仍可提交 | 增加上限 100000 字符，超长返回 400                              | 超长拒绝，正常长度仍可提交       |
| 4   | 用例 33          | 论文类人工审核详情缺少 AI 摘要与原文展示     | 前端/联调修复专家审核详情页                                       | 论文任务可查看摘要与原文        |
