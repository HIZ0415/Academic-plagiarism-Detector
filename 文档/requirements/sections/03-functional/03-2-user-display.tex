\subsubsection{功能点——个人信息展示（FR-YHZS-0001）}

\paragraph*{简述}

用户可通过该功能查看个人基本信息，包括昵称、ID、头像、等级等信息。

\paragraph*{前提条件}

用户已注册并登录系统，且系统数据库存储了该用户需要展示的信息。

\paragraph*{主要流程}

\begin{enumerate}
    \item 用户进入“个人中心”页面。
    \item 系统从数据库调取用户信息并渲染至前端页面。
    \item 用户可浏览个人基础信息，但不可直接修改（需通过其他模块操作）。
\end{enumerate}

\paragraph*{后继结果}

用户成功查看个人资料，页面显示完整信息。若数据加载失败，在该数据展示的位置提示“数据加载失败”。

\subsubsection{功能点——检测记录展示（FR-YHZS-0002）}

\paragraph*{简述}

用用户可按时间顺序查看一段时间内的图像检测任务记录，支持筛选、下载报告及删除操作。

\paragraph*{前提条件}

用户已登录。

\paragraph*{主要流程}

\begin{enumerate}
    \item 用户进入“检测记录”页面。
    \item 系统默认按时间倒序展示半年内的检测任务条目（即越新越靠前）。
    \item 用户可以通过时间范围或类别进行筛选。
    \item 用户可通过关键字进行查找。
    \item 点击任务条目可查看详情，并下载检测报告。
    \item 用户可选择删除指定记录，需二次确认后完成操作。
\end{enumerate}

\paragraph*{后继结果}

\begin{enumerate}
    \item 用户进入“检测记录”页面。
    \item 系统默认按时间倒序展示半年内的检测任务条目（即越新越靠前）。
    \item 用户可以通过时间范围或类别进行筛选。
    \item 用户可通过关键字进行查找。
    \item 点击任务条目可查看详情，并下载检测报告。
    \item 用户可选择删除指定记录，需二次确认后完成操作。
\end{enumerate}

\subsubsection{功能点——审核任务展示（FR-YHZS-0003）}

\paragraph*{简述}

用户可查看半年内提交的人工审核任务记录，支持查看当前详情、下载报告及终止审核。

\paragraph*{前提条件}

用户已提交过人工审核请求且审核任务数据已存储至系统数据库。

\paragraph*{主要流程}

\begin{enumerate}
    \item 用户进入“人工审核”页面。
    \item 系统按时间倒序展示用户提交的审核任务条目。
    \item 点击条目可查看审核详情及当前进度。
    \item 用户可下载已生成的审核报告，或选择终止未完成的审核任务。
\end{enumerate}

\paragraph*{后继结果}

若有人工审核任务，则用户成功查看审核任务状态及结果，若无，则显示“还没有检测记录”。终止操作后，任务状态更新为“已取消”，其他用户无法继续参与审核。

\subsubsection{功能点——社区反馈展示（FR-YHZS-0004）}

\paragraph*{简述}

用户可查看管理员对举报或评论的处理结果，以及个人审核任务、评论的新回复。

\paragraph*{前提条件}

用户曾参与过评论或举报操作。若用户参与过举报操作，要求管理员已完成相关处理并更新数据。

\paragraph*{主要流程}

\begin{enumerate}
    \item 用户进入“社区反馈”页面。
    \item 系统展示管理员对用户举报或评论的反馈信息。
    \item 用户可查看处理结果详情及新回复内容。
\end{enumerate}

\paragraph*{后继结果}

若有个人消息新回复或者举报处理情况更新，则标注为新消息，其余标注为老消息。若无消息，则展示“暂无消息”。
