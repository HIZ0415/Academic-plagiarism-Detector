\subsection{用户审核模块}

\paragraph*{业务流程}

用户登录系统后进入“人工审核”页面，可查看其他用户提交的人工审核任务，包括学术图像、整篇论文及同行评审Review的审核任务。

用户可参与审核、发表评论、点赞、举报违规内容，并提交审核结论。

\paragraph*{用例图}

\subsubsection{功能点——审核任务展示 （FR-YHSH-0001）}

\paragraph*{简述}

按照发布时间先后展示用户要求人工审核的任务条目，包括学术图像、论文及Review任务。

支持按照学科、任务类型、紧急程度、风险等级等分类查看审核任务；

支持详情页面展示已有AI检测结果和人工审核结果。

\paragraph*{前提条件}

由所有用户使用，用户必须已登录系统。

\paragraph*{主要流程}

业务流程为：

\begin{enumerate}
    \item 用户登录后进入“人工审核”页面；
    \item 系统展示任务列表；
    \item 用户选择筛选条件：
    \begin{enumerate}
        \item 学科分类
        \item 审核类型（图像 / 论文 / Review）
        \item 紧急程度
        \item 风险等级
    \end{enumerate}
    \item 系统分类显示审核任务条目；
    \item 用户点击任务条目查看详情；
    \item 页面展示AI检测结果、已有审核意见及当前审核进度。
\end{enumerate}

\paragraph*{后继结果}

用户可对审核任务进行审核、评论、点赞及举报操作。

\subsubsection{功能点——用户审核（FR-YHSH-0002）}

\paragraph*{简述}

用户可对他人提交的人工审核任务进行审查，包括学术图像、论文文本及Review内容，提交鉴定结论及理由。

系统提供AI检测报告作为审核参考。

\paragraph*{前提条件}

用户已进入审核任务详情页面。

输入信息：鉴定结论及其理由。

\paragraph*{主要流程}

业务流程为：

\begin{enumerate}
    \item 用户进入审核任务详情页面；
    \item 点击“查看报告”按钮查看AI检测报告；
    \item 选择鉴定结论：
    \begin{enumerate}
        \item 确认造假
        \item 疑似造假
        \item 未发现异常
    \end{enumerate}
    \item 填写文字理由；
    \item 点击“提交审核”；
    \item 系统保存审核意见并更新审核进度。
\end{enumerate}

\paragraph*{后继结果}

输出信息：

发起审核用户收到结果通知；

系统更新审核统计结果。

\subsubsection{功能点——评论点赞（FR-YHSH-0003）}

\paragraph*{简述}

用户可对审核任务发表评论或点赞，促进学术交流。

系统支持过滤敏感词；

支持按点赞数、发表时间排序评论。

\paragraph*{前提条件}

用户已进入审核任务详情页面。

输入信息：用户评论或点赞。

\paragraph*{主要流程}

业务流程为：

\begin{enumerate}
    \item 用户进入审核任务详情页面；
    \item 点击点赞按钮进行点赞；
    \item 或点击评论输入框发表评论；
    \item 系统进行敏感词过滤；
    \item 评论成功后显示在评论区。
\end{enumerate}

\paragraph*{后继结果}

输出信息：

发起审核用户收到互动通知。

\subsubsection{功能点——用户举报（FR-YHSH-0004）}

\paragraph*{简述}

针对违规审核任务和评论，用户可选择举报，并说明情况交由管理端处理。

\paragraph*{前提条件}

用户发现违规内容。

输入信息：举报类型、举报理由。

\paragraph*{主要流程}

业务流程为：

\begin{enumerate}
    \item 用户进入审核任务详情页面；
    \item 点击违规任务或评论旁“举报”按钮；
    \item 选择举报类型；
    \item 填写举报理由；
    \item 提交举报；
    \item 系统自动进入管理端举报处理队列。
\end{enumerate}

\paragraph*{后继结果}

输出信息：

举报用户收到举报处理结果反馈；

违规用户可能被禁言、限制使用或封号。
