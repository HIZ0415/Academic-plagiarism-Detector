\subsection{用户上传模块}

\paragraph*{业务流程}

用户登录系统后，可进入“学术内容检测”页面，上传待检测的学术资源，包括学术图像、整篇论文文件、同行评审Review文本及其压缩包文件。

系统完成文件合法性校验后，将任务提交至AI检测模块进行自动检测，并实时同步检测状态。

用户可查看检测结果、下载综合鉴伪报告，并根据需要申请人工审核。

\paragraph*{用例图}

\subsubsection{功能点——文件上传（FR-YHSC-0001）}

\paragraph*{简述}

用户登录后进入检测页面，上传待检测的学术图像、整篇论文、同行评审Review文本文件、包含学术资源的文件或其压缩包。

\paragraph*{前提条件}

由所有用户使用，用户必须已登录系统。

输入信息：用户选择的本地文件或压缩包。

支持格式包括：

PNG、JPG、JPEG、PDF、DOCX、TXT、ZIP、RAR 等格式。

\paragraph*{主要流程}

业务流程为：

\begin{enumerate}
    \item 用户登录系统后进入“学术检测”页面；
    \item 点击“上传文件”按钮；
    \item 选择本地待检测文件；
    \item 系统自动识别文件类型（图像 / 论文 / Review）；
    \item 校验文件格式、大小及完整性；
    \item 页面显示上传进度条；
    \item 上传成功后自动跳转至“检测结果”页面。
\end{enumerate}

异常情况下：

若文件格式错误、文件损坏、大小超限或上传中断，则提示失败信息。

\paragraph*{后继结果}

输出信息：

上传成功提示；

自动跳转至检测结果页；

或上传失败提示。

\subsubsection{功能点——AI检测（FR-YHSC-0002）}

\paragraph*{简述}

用户上传待检测文件后，系统自动执行AI检测，并即时返回图像、论文或Review的检测结果，支持查看详细可视化分析报告及下载综合鉴伪报告。

\paragraph*{前提条件}

用户上传文件成功。

\paragraph*{主要流程}

业务流程为：

\begin{enumerate}
    \item 文件上传成功后，系统自动创建检测任务；
    \item 用户进入“检测结果”页面；
    \item 系统执行AI检测：
    \begin{enumerate}
        \item 图像造假检测；
        \item 论文AIGC文本检测；
        \item Review虚假评审检测；
    \end{enumerate}
    \item 页面实时展示检测进度；
    \item 检测完成后展示鉴定结果；
    \item 用户点击“查看报告”按钮查看详细可视化分析报告；
    \item 支持下载综合鉴伪报告。
\end{enumerate}

异常情况下：

若AI检测服务异常，则提示“AI检测异常”，不提供报告下载功能。

\paragraph*{后继结果}

输出信息：

用户可下载 PDF / HTML 格式综合鉴伪报告。

检测记录自动保存至用户检测任务详情页。

\subsubsection{功能点——人工审核请求（FR-YHSC-0003）}

\paragraph*{简述}

用户对AI检测结果存在疑问时，可提交人工审核请求，将当前检测任务提交至人工审核模块。

\paragraph*{前提条件}

AI检测已完成。

\paragraph*{主要流程}

\begin{enumerate}
    \item 用户进入检测结果页面；
    \item 若对结果不满意，点击“申请人工审核”按钮；
    \item 系统提交当前文件及AI检测结果至人工审核模块；
    \item 系统生成审核任务编号；
    \item 跳转至人工审核详情页面。
\end{enumerate}

异常情况下：

若提交失败，则提示失败信息。

\paragraph*{后继结果}

输出信息：

提交成功提示；

审核任务自动进入人工审核页面。

\subsubsection{功能点——批量上传（FR-YHSC-0004）}

\paragraph*{简述}

支持用户一次性上传多个学术资源文件，实现批量检测任务创建。

\paragraph*{前提条件}

用户已登录系统。

\paragraph*{主要流程}

\begin{enumerate}
    \item 用户点击“批量上传”按钮；
    \item 选择多个文件或压缩包；
    \item 系统逐个校验文件合法性；
    \item 批量创建检测任务；
    \item 返回批量任务列表页面。
\end{enumerate}

\paragraph*{后继结果}

输出信息：

生成批量检测任务；

显示任务数量及状态。

\subsubsection{功能点——高并发批量检测（FR-YHSC-0005）}

\paragraph*{简述}

系统支持高并发场景下的大规模批量检测任务处理。

\paragraph*{前提条件}

批量上传任务已创建。

\paragraph*{主要流程}

\begin{enumerate}
    \item 系统将批量任务分配至多线程任务队列；
    \item 后台并行执行图像、论文及Review检测；
    \item 系统动态分配计算资源；
    \item 异常任务自动重试。
\end{enumerate}

\paragraph*{后继结果}

输出信息：

保证高并发下任务稳定执行；

任务结果统一汇总。

\subsubsection{功能点——检测状态实时同步（FR-YHSC-0006）}

\paragraph*{简述}

系统实时同步检测状态，便于用户动态查看任务进度。

\paragraph*{前提条件}

检测任务已创建。

\paragraph*{主要流程}

\begin{enumerate}
    \item 用户进入任务详情页面；
    \item 系统实时刷新任务状态；
    \item 展示状态：
    \begin{enumerate}
        \item 待检测
        \item 检测中
        \item 人工审核中
        \item 已完成
        \item 检测失败
    \end{enumerate}
    \item 状态变化自动推送至前端。
\end{enumerate}

\paragraph*{后继结果}

输出信息：

用户实时查看检测进度及当前状态。
