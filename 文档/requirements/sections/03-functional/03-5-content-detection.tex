\subsubsection{功能点——提取图片（FR-TPJC-0001）}

\paragraph*{简述}

从用户上传的 PDF、压缩包及论文文件等非图像格式文件中自动抽取学术图像，便于后续进行图像AI鉴伪检测。

该功能旨在自动化处理多种文件格式，确保图像检测流程顺利进行。

系统能够快速提取高质量图像数据，为后续AI算法提供输入。

\paragraph*{前提条件}

用户上传文件格式为 PDF、ZIP、RAR、DOCX、TAR、7Z 等非图像格式。

文件大小不超过系统规定上限（如 500MB）。

文件中至少包含一张图像。

\paragraph*{主要流程}

\begin{enumerate}
    \item 用户上传文件；
    \item 系统自动识别文件格式；
    \item 调用对应解析工具：
    \begin{enumerate}
        \item PDF解析库
        \item 压缩文件解压工具
        \item DOCX文档解析工具
    \end{enumerate}
    \item 对PDF逐页提取嵌入图像；
    \item 对压缩文件递归遍历子目录提取图像；
    \item 对论文文档抽取正文插图、实验图及附录图像；
    \item 图像预处理：
    \begin{enumerate}
        \item 格式统一为 JPG / PNG
        \item 分辨率标准化
        \item 色彩空间统一
    \end{enumerate}
    \item 临时存储；
    \item 生成图像文件列表。
\end{enumerate}

\paragraph*{后继结果}

系统输出结构化图像列表，包含：

\begin{enumerate}
    \item 文件名
    \item 格式
    \item 分辨率
    \item 来源页码
    \item 图像编号
\end{enumerate}

图像列表以 JSON 格式存储。

\paragraph*{补充说明}

支持批量文件处理；

支持异常提示：

文件损坏、格式不支持、解析失败。
}

\subsubsection{功能点——学术图像AI鉴伪（FR-TPJC-0002）}

\paragraph*{简述}

基于深度学习和图像处理技术，对提取图像进行自动化学术造假检测。

检测内容包括：

\begin{enumerate}
    \item 美化修饰
    \item 背景掩盖
    \item 复制伪造
    \item 跨图拼接
    \item AI生成图像识别
\end{enumerate}

系统能够高效识别图像异常区域并输出可解释结果。

\paragraph*{前提条件}

系统已成功提取图像数据；

图像已完成预处理。

\paragraph*{主要流程}

\begin{enumerate}
    \item 图像预处理：
    \begin{enumerate}
        \item 尺寸归一化
        \item 色彩归一化
        \item 噪声去除
    \end{enumerate}
    
    \item 特征提取：
    \begin{enumerate}
        \item CNN深度特征
        \item 边缘纹理特征
        \item 频域特征
    \end{enumerate}
    
    \item 图像鉴伪检测：
    \begin{enumerate}
        \item 美化修饰检测
        \item 背景掩盖检测
        \item 同图复制检测
        \item 跨图拼接检测
        \item AI生成痕迹检测
    \end{enumerate}
    
    \item 计算风险评分；
    
    \item 输出可疑区域坐标；
    
    \item 生成检测报告。
\end{enumerate}

\paragraph*{后继结果}

系统输出检测报告，包括：

\begin{enumerate}
    \item 原图
    \item 标注图
    \item 可疑区域说明
    \item 造假类型
    \item 风险等级
    \item 置信度评分
\end{enumerate}

报告支持 JSON / PDF / HTML 格式。

\paragraph*{补充说明}

支持单图检测与批量检测；

支持综合鉴伪报告生成；

支持可视化展示。
}

\begin{enumerate}
    \item 异常区域框选
    \item 区域编号
    \item 风险等级
    \item 类型说明
\end{enumerate}

支持用户下载标注图。

\paragraph*{补充说明}

支持多区域同时标注；

支持细粒度像素级区域高亮；

支持人工审核二次标记。
}


\subsubsection{\iternew{功能点——全篇论文AIGC检测（FR-LWJC-0001）}}

\paragraph*{简述}

\iternew{系统应支持对用户上传的全篇论文进行 AIGC 检测，并输出论文整体层面的检测结论。该功能应面向整篇论文文本进行统一分析，在结果中给出全文级风险判断、AI 贡献占比或等价指标、任务状态及结果摘要信息，为后续的段落级识别、概率分析和事实性鉴伪提供基础。}

\paragraph*{前提条件}

用户已登录并具备论文检测权限；用户已按系统支持的格式成功上传论文内容；系统已完成检测任务创建，且论文文本可被正常解析。

\paragraph*{主要流程}

\begin{enumerate}
    \item 用户在论文检测页面上传论文文件或提交论文文本，发起全篇论文 AIGC 检测任务。
    \item 系统对上传内容进行格式校验、文本解析与结构化预处理，生成对应的论文检测任务。
    \item 系统调用论文 AIGC 检测模块，对整篇论文进行统一分析，形成全文级检测结果。
    \item 系统向用户返回任务受理状态；当检测完成后，展示全文级检测结论、AI 贡献占比或等价指标、结果摘要及更新时间等信息。
    \item 用户可基于该结果继续查看 AI 生成段落识别结果、段落级概率分析结果以及事实性鉴伪相关结果。
\end{enumerate}

\paragraph*{后继结果}

系统形成并保存论文全文级检测记录；用户能够从整篇论文层面快速了解该论文是否存在较高的 AI 生成风险，并可继续查看更细粒度的分析结果。若检测失败、文本解析异常或任务处理中，系统应返回明确状态说明。

\subsubsection{\iternew{功能点——AI生成段落识别（FR-LWJC-0002）}}

\paragraph*{简述}

\iternew{在完成全篇论文检测后，系统应能够自动识别论文中疑似由 AI 生成的文本段落，并以结构化方式返回相应段落内容、段落位置或段落编号，以支持用户对风险内容进行精确定位和审阅。}

\paragraph*{前提条件}

对应论文检测任务已成功创建；系统已完成论文分段或分片处理；用户具备查看该检测任务详细结果的权限。

\paragraph*{主要流程}

\begin{enumerate}
    \item 系统对论文正文进行段落划分，并结合论文 AIGC 检测模型识别其中疑似 AI 生成的文本段落。
    \item 系统提取被识别段落的段落编号、文本内容、原文位置或等价定位信息，并形成段落级结果列表。
    \item 系统在检测结果页面中以列表、高亮或其他清晰方式展示疑似 AI 生成段落，支持用户按段落逐条查看。
    \item 用户可点击对应条目定位至原文位置，并结合段落级概率和事实性鉴伪信息进行进一步分析。
\end{enumerate}

\paragraph*{后继结果}

用户能够明确获知论文中哪些段落被系统识别为疑似 AI 生成内容，并可基于段落定位信息快速完成核查。若未识别出疑似 AI 生成段落，系统应返回明确的空结果说明。

\subsubsection{\iternew{功能点——段落级AI生成概率分析（FR-LWJC-0003）}}

\paragraph*{简述}

\iternew{对于论文中的疑似 AI 生成段落或系统纳入分析范围的全部段落，系统应给出段落级 AI 生成概率或等价风险分值，以反映不同段落的 AI 生成倾向强弱，为用户提供更细粒度的判断依据。}

\paragraph*{前提条件}

系统已完成段落切分与段落级检测；检测模型能够输出段落级 AI 生成概率、风险分值或风险等级字段；用户具备查看该任务详细结果的权限。

\paragraph*{主要流程}

\begin{enumerate}
    \item 系统对论文段落逐段计算 AI 生成概率、风险分值或等价风险等级。
    \item 系统将概率结果与具体段落进行一一对应，并形成结构化的段落级分析结果。
    \item 系统在检测结果页面中展示每个段落对应的概率信息，必要时支持排序、筛选或重点段落标识。
    \item 用户可结合段落识别结果查看高风险段落，并进一步查看事实性鉴伪子结论与原因说明。
\end{enumerate}

\paragraph*{后继结果}

用户能够从段落粒度理解系统对论文内容的判断强度，并识别出高风险段落。系统应保证段落标识与概率信息一一对应，避免出现结果错位或语义不清的情况。

\subsubsection{\iternew{功能点——事实性鉴伪子结论生成（FR-LWJC-0004）}}

\paragraph*{简述}

\iternew{在论文 AIGC 检测基础上，系统应能够针对存在风险的论文段落或内容片段生成事实性鉴伪子结论。子结论用于概括系统在局部内容上识别出的可疑问题或风险类型，为用户提供比“整篇论文是否可疑”更具体的分析结果。}

\paragraph*{前提条件}

系统已完成全文检测、段落识别与段落级概率分析；对应任务已启用事实性鉴伪能力；后端能够输出子结论结构化结果。

\paragraph*{主要流程}

\begin{enumerate}
    \item 系统综合全文级检测结果、段落级检测结果及事实性分析结果，对论文中的风险内容进行归纳。
    \item 系统为相关段落或内容片段生成对应的事实性鉴伪子结论，并建立子结论与原文位置之间的映射关系。
    \item 系统按照条目化或分组化方式展示事实性鉴伪子结论，使用户能够逐条查看各项子结论的内容。
    \item 用户可进一步展开查看每条子结论对应的原因说明及相关依据。
\end{enumerate}

\paragraph*{后继结果}

系统形成面向论文局部内容的事实性鉴伪子结论集合，并与对应段落建立关联；用户能够从结构化结果中理解论文中具体存在何种可疑风险，而非仅获得单一总体结论。

\subsubsection{\iternew{功能点——事实性鉴伪原因说明（FR-LWJC-0005）}}

\paragraph*{简述}

\iternew{对于每一条事实性鉴伪子结论，系统应提供相应的原因说明，用于解释该子结论形成的依据、判定逻辑或相关证据要点，以提升检测结果的可理解性和可审查性。}

\paragraph*{前提条件}

系统已生成事实性鉴伪子结论；后端可返回与子结论对应的原因说明、证据摘要、关联段落或其他解释性字段；用户具备查看相关结果的权限。

\paragraph*{主要流程}

\begin{enumerate}
    \item 系统针对每条事实性鉴伪子结论生成对应的原因说明，并与子结论条目建立关联。
    \item 系统在检测结果页面中为每条子结论提供展开查看入口，展示其原因说明、依据要点或关联片段。
    \item 用户结合原文段落、段落级概率及子结论内容，对系统给出的原因说明进行阅读和比对。
    \item 如系统支持综合鉴伪报告下载，则相应原因说明应同步写入报告内容中。
\end{enumerate}

\paragraph*{后继结果}

用户能够理解各条事实性鉴伪子结论的形成依据，从而提升对检测结果的信任度与可审阅性。若个别子结论暂不具备充分说明内容，系统应给出明确提示，避免出现无解释结果。

\subsubsection{\iternew{功能点——同行评审Review检测（FR-PLJC-0001）}}

\paragraph*{简述}

\iternew{系统针对用户上传的同行评审（Review）文本内容进行自动化AI检测，识别其中是否存在AI生成、虚假评审或异常文本特征。}

该功能主要用于辅助期刊编辑、科研管理人员及用户识别虚假同行评审行为，维护学术评审的真实性和可信度。

\paragraph*{前提条件}

用户已成功上传Review文本文件或输入Review文本内容。

支持格式包括：

TXT、DOCX、PDF、在线文本输入。

\paragraph*{主要流程}

\begin{enumerate}
    \item 用户上传同行评审文本；
    \item 系统自动进行文本预处理：
    \begin{enumerate}
        \item 去除格式符号
        \item 分句与分词
        \item 语言标准化
    \end{enumerate}
    \item 调用NLP检测模型；
    \item 分析文本特征：
    \begin{enumerate}
        \item 语言流畅度
        \item 语义重复率
        \item AI生成概率
        \item 情感倾向异常
    \end{enumerate}
    \item 识别是否存在AI生成痕迹；
    \item 输出风险评分及检测结论；
    \item 保存检测记录。
\end{enumerate}

\paragraph*{后继结果}

输出检测结果，包括：

\begin{enumerate}
    \item 是否疑似AI生成
    \item 风险等级
    \item 可疑段落位置
    \item 置信度评分
\end{enumerate}

检测结果自动进入用户检测任务记录。

\paragraph*{补充说明}

支持批量Review检测；

支持英文及中文评审文本检测。

\subsubsection{\iternew{功能点——模板化评审倾向识别（FR-PLJC-0002）}}

\paragraph*{简述}

\iternew{系统对同行评审文本进行模板化倾向分析，识别是否存在高度重复、机械化或批量生成的评审内容。}

主要用于发现批量刷评审、虚假同行评审及低质量模板化意见。

\paragraph*{前提条件}

同行评审Review检测任务已创建。

\paragraph*{主要流程}

\begin{enumerate}
    \item 获取Review文本内容；
    \item 提取语义特征和句式结构特征；
    \item 分析以下内容：
    \begin{enumerate}
        \item 固定句式重复率
        \item 常见模板短语匹配
        \item 历史评审相似度比对
        \item 内容长度异常
    \end{enumerate}
    \item 判断是否存在模板化倾向；
    \item 输出模板化风险评分；
    \item 标注可疑语句。
\end{enumerate}

\paragraph*{后继结果}

输出信息包括：

\begin{enumerate}
    \item 模板化倾向等级
    \item 高重复语句位置
    \item 历史相似文本匹配结果
    \item 综合风险评分
\end{enumerate}

\paragraph*{补充说明}

支持跨历史数据库相似评审比对；

支持多语言模板检测扩展。

\subsubsection{功能点——综合鉴伪结果生成（FR-NRJC-0001）}\paragraph*{简述}

系统整合图像检测、论文检测、同行评审检测及人工审核结果，自动生成综合鉴伪报告。

该报告用于向用户提供完整、可解释的最终检测结果。

\paragraph*{前提条件}

至少完成以下任一检测任务：

\begin{enumerate}
    \item 图像AI鉴伪
    \item 论文AIGC检测
    \item Review检测
    \item 人工审核
\end{enumerate}

\paragraph*{主要流程}

\begin{enumerate}
    \item 汇总所有检测子模块结果；
    \item 统一风险评分标准；
    \item 综合计算最终风险等级；
    \item 汇总：
    \begin{enumerate}
        \item 图像可疑区域
        \item 可疑文本段落
        \item Review异常内容
        \item 人工审核意见
    \end{enumerate}
    \item 自动生成综合鉴伪报告；
    \item 支持PDF / HTML导出。
\end{enumerate}

\paragraph*{后继结果}

输出综合鉴伪报告，包括：

\begin{enumerate}
    \item 综合风险等级
    \item 总体结论
    \item 各子模块详细结果
    \item 人工审核建议
    \item 使用风险提示
\end{enumerate}

报告自动保存至用户历史记录页面。

\paragraph*{补充说明}

支持多任务合并报告；

支持后续追加检测结果动态更新。



\subsubsection{功能点——检测结果可视化解释（FR-NRJC-0002）}

\paragraph*{简述}

系统应支持对检测结果进行可视化解释，通过段落高亮、概率分布、图像可疑区域标注、标签化提示或其他可视化方式，展示 AI 判定“造假”或存在风险的依据，提升鉴伪结果的逻辑透明化水平，帮助用户更直观地理解系统结论。

\paragraph*{前提条件}

系统已完成相应检测任务并生成结构化结果；后端已返回可用于可视化展示的数据，包括但不限于段落位置、段落级概率、事实性鉴伪条目、图像可疑区域坐标或 Review 风险标签等；用户具备查看该任务结果的权限。

\paragraph*{主要流程}

\begin{enumerate}
    \item 用户进入检测结果详情页或综合鉴伪报告查看页。
    \item 系统依据检测对象类型加载对应的可视化解释组件：对于论文，可展示疑似 AI 段落高亮、段落级概率分布和事实性鉴伪关联信息；对于图像，可展示 AI 可疑区域的细粒度标注；对于 Review，可展示 AI 生成或模板化倾向的结构化提示信息。
    \item 用户可通过点击、悬停、筛选或切换视图等方式查看局部解释信息，并将可视化结果与文字结论进行对照。
    \item 系统应保证可视化解释与综合鉴伪结论、文本说明和原始检测结果保持一致，必要时同步展示使用建议或风险提示。
\end{enumerate}

\paragraph*{后继结果}

用户能够基于可视化信息更直观地理解系统给出的检测依据，提高结果解释性和可审查性。若某类结果暂不具备可视化展示条件，系统应采用结构化列表或文字说明进行降级展示，并保持结论表达清晰一致。

\subsubsection{\iternew{功能点——多模态联合检测（FR-NRJC-0003）}}

\paragraph*{简述}

\iternew{在保留原有论文文本检测、图像检测及 Review 检测功能的基础上，}
\iternew{系统新增多模态联合检测能力，}
\iternew{对同一任务中的多类型学术内容进行统一分析。}

\paragraph*{检测内容包括：}

\begin{enumerate}
    \item 论文正文文本 AI 生成倾向检测；
    \item 学术图像真实性检测；
    \item 图像局部篡改与拼接分析；
    \item 同行评审 Review 模板化检测；
    \item Review AI 生成概率分析；
    \item 跨模态一致性校验。
\end{enumerate}

\paragraph*{后继结果：}

输出多维度检测子结果，
并进入综合结果分析流程。

\subsubsection{\iternew{功能点——综合结果融合分析（FR-NRJC-0004）}}

\paragraph*{简述：}

\iternew{系统在完成文本、图像及 Review 各维度检测后，}
\iternew{对检测结果进行统一融合分析，}
\iternew{形成综合鉴伪结论。}

\paragraph*{分析内容包括：}

\begin{enumerate}
    \item 综合可信度评分；
    \item 风险等级划分；
    \item 分维度风险说明；
    \item 可疑内容定位；
    \item 人工复核建议；
    \item 综合鉴伪结论生成。
\end{enumerate}

\paragraph*{后继结果：}

生成统一综合检测报告，
支持用户查看与下载。
