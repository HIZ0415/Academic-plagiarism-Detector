\section{概述}

\subsection{简述}

近年来，学术界不断爆出学术造假事件，尤其是图像篡改问题，给学术环境带来了巨大冲击。学术图像造假行为不仅影响研究成果的真实性，还可能对科研人员的声誉和期刊的公信力造成严重影响。为了维护学术诚信、提升期刊质量并提高审稿效率，亟需开发一种高效的学术图像检测工具。本项目旨在通过开发学术图像造假检测系统，结合AI自动检测和人工审查的双重保障，帮助科研人员和期刊管理者快速准确地识别学术图像中的造假行为，避免因图像问题导致的学术争议，促进科研工作的健康发展。

系统将允许用户上传各类图像文件，通过AI算法进行自动化检测，并结合人工审核进一步确认图像的真实性。平台提供全面的功能支持，包括个人信息展示、图像上传、检测记录查询、人工审核请求、图像检测等，致力于为科研人员、期刊审稿人及学术机构提供一站式服务，从而提高学术图像检测的准确性和效率。

\subsection{用户群体}

\subsubsection{普通用户}

普通用户是系统的主要用户群体，包括科研人员、学术期刊编辑、学生等。他们的需求主要集中在学术图像的检测和识别上，确保上传的学术图像未经过篡改。普通用户将通过平台享受以下功能：

\begin{itemize}
    \item 上传待检测图像，并获取AI自动检测结果，下载检测报告；
    \item 请求人工审核以提高检测结果的准确性；
    \item 查看历史上传记录和检测报告；
    \item 获取平台的社区反馈，参与评论、点赞等互动功能；
    \item 对需要人工审核的图片进行审核，给出个人建议。
\end{itemize}

\subsubsection{管理员用户}

管理员用户负责平台的管理和维护，具有较高的权限，可以执行以下操作：

\begin{itemize}
    \item 管理用户和检测任务，包括修改、删除用户信息，处理举报等；
    \item 维护平台的运营，执行数据统计、操作记录分析等任务；
    \item 审核用户举报的内容，并根据需要采取措施，如警告或封号。
\end{itemize}

\subsubsection{超级管理员用户}

超级管理员除具有普通管理员的所有权限外，还能够对普通管理员用户账号进行管理，即可执行：

\begin{itemize}
    \item 对普通管理员进行添加、删除及权限分配操作。
\end{itemize}

\subsection{系统范围}

\subsubsection{用户通用功能}

\begin{itemize}
    \item 注册与登录：用户可以注册并登录平台，管理个人信息；
    \item 图像上传与检测：用户可上传图像进行AI自动检测，获取检测结果；
    \item 人工审核请求：用户可选择是否要求人工审核，以提高检测准确性；
    \item 历史记录查询：用户可以查看历史上传的检测任务，查看报告并下载；
    \item 社区互动：支持用户对检测任务和评论进行点赞、评论和举报。
    \item 人工审核：用户可对申请人工审核的图片进行审核并给出个人意见，确保准确性。
\end{itemize}

\subsubsection{管理员功能}

\begin{itemize}
    \item 用户管理：管理员可对用户信息进行修改、删除、查询等操作；
    \item 数据统计与分析：管理员可查看平台的运行数据，进行必要的统计与分析；
    \item 举报处理：管理员处理用户的举报，针对恶意行为采取适当措施；
    \item 人工审核管理：管理员可对人工审核请求进行管理，确保检测结果的公正性和准确性。
\end{itemize}

\subsubsection{超级管理员功能}

\begin{itemize}
    \item 管理员用户管理：超级管理员可对普通管理员进行添加、删除及权限分配操作。
\end{itemize}
