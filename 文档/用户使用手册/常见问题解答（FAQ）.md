# 常见问题解答（FAQ）

> **适用系统**：学术内容诚信检测系统（用户端 / 管理端）  
> **文档版本**：1.0  
> **更新日期**：2026-05-24  
> **说明**：本节汇总平台使用过程中的高频问题，按角色与功能模块分类。若问题涉及本地部署或开发者联调，请参阅《本地启动与停止手册》等运维文档。

---

## 一、术语与角色说明

### Q1. 「Review 检测」和「人工审核」是一回事吗？

**不是，请严格区分：**


| 名称            | 含义                               | 入口                                 |
| ------------- | -------------------------------- | ---------------------------------- |
| **Review 检测** | 对评审意见（Review）文本进行 **AI 自动鉴伪**    | 用户端 → 学术检测 → **Review 检测** 标签      |
| **人工审核**      | 编辑发起申请，由组织管理员审批、专家复核的 **人工鉴定流程** | 用户端 → **人工审核申请**（编辑）/ **人工审核**（专家） |


Review 检测是自动检测功能；人工审核是「申请 → 审批 → 专家鉴定 → 结果汇总」的闭环流程，两者相互独立，也可在检测完成后对同一任务发起人工审核。

### Q2. 系统有哪些用户角色？各自能做什么？


| 角色          | 登录入口         | 主要权限                                     |
| ----------- | ------------ | ---------------------------------------- |
| **编辑（发布者）** | 用户端，登录页选「编辑」 | 学术检测、检测历史、综合鉴伪报告、发起人工审核申请、社区反馈           |
| **专家（审稿人）** | 用户端，登录页选「专家」 | 人工审核任务池、提交鉴定结论、个人主页                      |
| **组织管理员**   | 管理端          | 人工审核审批（通过/拒绝）、本组织用户与资源管理、系统日志            |
| **软件管理员**   | 管理端          | 跨组织治理、检测任务监控、模型配置、举报处理；**不能**审批本组织人工审核申请 |


### Q3. 「标准模式」和「精准模式」有什么区别？

- **标准模式**：检测速度较快，适合日常批量筛查。
- **精准模式**：检测策略更严格，部分能力依赖 AI 服务是否启用 LLM；界面已支持切换，实际效果以当前部署环境为准。

两种模式仅用于 **学术检测（AI 鉴伪）**，与登录页选择的「编辑 / 专家」角色无关。

## 二、登录与账号

### Q4. 登录时提示验证码错误或账号密码错误怎么办？

1. **图形验证码区分大小写**，请重新输入或刷新验证码后再试。
2. 确认邮箱与密码无误；编辑与专家使用 **不同测试/正式账号**，不可混用。
3. 登录页须先选择正确身份：**编辑** 或 **专家**，选错可能导致后续功能不可用。
4. 若多次失败，请联系组织管理员确认账号是否已开通、是否已绑定组织。

### Q5. 编辑和专家可以在同一个浏览器里切换登录吗？

可以，但须 **先退出当前账号再登录另一角色**。  
不建议在同一浏览器会话中混用编辑与专家账号，否则可能出现菜单显示异常或权限跳转错误。

### Q6. 如何注册新账号？

1. 打开用户端登录页，点击 **注册**。
2. 填写邮箱、密码等信息，并输入 **邀请码**（由组织管理员发放）。
3. 邀请码决定注册为 **编辑** 或 **专家**，不可通过注册获得管理端管理员权限。
4. 注册成功后，使用对应身份登录用户端即可。

> **注意**：账号注册在用户端完成，不在管理端注册。

### Q7. 专家登录后为什么看不到「学术检测」菜单？

这是 **正常现象**。专家账号仅开放 **人工审核** 与 **个人主页**；若访问编辑侧路径（如学术检测、人工审核申请），系统会自动跳转到专家任务池（`/review`）。

### Q8. 编辑登录后为什么进不了专家审核页面？

编辑账号无权访问专家任务池。若误访问 `/review` 等路径，系统会自动跳转到 **学术检测** 页面。

## 三、学术检测

### Q9. 学术检测入口在哪里？包含哪些类型？

侧栏 **学术检测**（路径 `/upload`）为统一入口，内含三个标签页：


| 标签            | 用途                             |
| ------------- | ------------------------------ |
| **批量提交**      | 上传图像、压缩包，或混合提交 PDF / Review 文本 |
| **论文检测**      | 上传论文 PDF，进行 AIGC 检测与学术资源规范性检查  |
| **Review 检测** | 在线输入或上传 `.txt` 文件，对评审文本进行鉴伪    |


### Q10. 支持哪些文件格式？


| 检测类型      | 支持格式              | 不支持 / 限制                    |
| --------- | ----------------- | --------------------------- |
| 图像检测      | PNG、JPG 等常见图片格式   | 不支持 `.exe` 等可执行文件           |
| 批量压缩包     | ZIP、RAR（自动抽取包内图片） | —                           |
| 论文检测      | **PDF**           | 不支持 DOCX 直传；请先将 Word 转为 PDF |
| Review 检测 | 在线文本、`.txt` 文件    | 文本不能为空；长度上限 **100,000 字符**  |


### Q11. 上传 DOCX 论文会怎样？

在批量提交入口上传 DOCX 时，系统会 **提示转换为 PDF**，**不会** 创建论文检测任务。请使用 PDF 后在「论文检测」标签页重新上传。

### Q12. 提交检测后任务一直显示「进行中」或失败怎么办？

常见原因与处理：

1. **检测服务繁忙或未就绪**：稍等片刻后刷新「检测历史」；若长时间无变化，请联系管理员确认 AI 检测服务是否在线。
2. **文件不符合要求**：如论文非 PDF、Review 文本为空或超长、PDF 损坏/加密无法解析等，任务会标记为失败。
3. **查看失败原因**：在检测历史的任务详情或列表中查看 **失败原因** 说明；管理端可在 **系统日志 → 检测任务记录** 中查看更详细错误信息。
4. **不要重复盲目提交**：失败请先根据提示修正文件或参数；「再次检测」会 **新建任务**，不会覆盖原任务。

### Q13. 标准模式和精准模式应该选哪个？

- 日常筛查、批量任务：建议 **标准模式**。
- 对单份重要材料需要更严格分析：可尝试 **精准模式**。
- 论文 AIGC 检测、Review 检测等异步任务，提交后请在 **检测历史** 中等待完成。

### Q14. Review 文本提交失败，提示内容不能为空或超过长度限制？

- 在线输入或 `.txt` 文件内容 **不能为空白**（含仅空格、换行）。
- 单条 Review 文本上限为 **100,000 字符**（约 10 万字），超出请分段或精简后重新提交。
- 含 BOM、控制字符的文本系统会自动清洗；若仍失败，请检查文件编码是否为 UTF-8。

### Q15. 论文 AIGC 检测结果如何理解？

论文检测完成后，可在检测历史详情中查看：

- **段落级 AIGC 风险**：各段落 AI 生成可能性及风险等级。
- **事实性鉴伪**：对论文陈述的事实性核查结论。
- **摘录高亮**：点击段落表某行可高亮对应摘录内容。

具体指标含义以结果页说明为准；如需权威认定，可进一步发起 **人工审核**。

## 四、检测历史与综合鉴伪报告

### Q16. 在哪里查看历史检测记录？

侧栏 **检测历史**（`/history`）可查看全部任务，支持按 **状态、类型、时间** 筛选（如「最近半年」）。

### Q17. 如何查看或下载单份检测报告？

1. 进入 **检测历史**，找到状态为 **已完成** 的任务。
2. 点击 **任务详情** 进入任务详情页。
3. 在详情页可查看 AI 检测结果摘要，并 **下载 PDF 检测报告**。
4. 未完成的任务无法下载报告，系统会提示「任务尚未完成」。

### Q18. 什么是「综合鉴伪报告」？如何查看？

**综合鉴伪报告** 将同一批次或关联任务中 **图像、论文、Review** 等多模态检测结果汇总为一份报告，便于整体评估。

- **在线查看**：检测历史 → 已完成任务 → **综合报告**（或访问 `/comprehensive-report`）。
- **下载 PDF**：在综合鉴伪报告页点击 **下载 PDF**。
- **导出 HTML**：在综合鉴伪报告页点击 **导出 HTML**，便于离线浏览或归档。

### Q19. 综合鉴伪报告页面空白或缺少某模态内容？

请确认：

1. 关联检测任务均已 **完成**，且无失败项。
2. 该批次确实包含对应类型任务（如无论文任务则不会出现论文小节）。
3. 刷新页面或从检测历史重新进入；仍异常请联系管理员。

## 五、人工审核（编辑）

### Q20. 什么情况下需要发起人工审核？

当自动检测结果需要 **专家复核、组织审批** 或形成正式鉴定结论时，编辑可在检测完成后发起人工审核申请。适用于图像、论文、Review 等各类已完成检测任务。

### Q21. 如何发起人工审核申请？

**方式一（推荐）**

1. 进入 **检测历史**，打开 **已完成** 任务的详情页。
2. 点击 **发起人工审核**（或类似入口），系统自动关联当前任务。

**方式二**

1. 侧栏进入 **人工审核申请**（`/annual`）。
2. 点击 **发起人工审核申请**，手动填写：
  - **关联自动检测任务 ID**（在检测历史中可查）
  - **任务类型**（须与检测任务一致，如图像检测、论文 AIGC、Review 检测等）
  - **申请理由**（不少于约 10 字）

### Q22. 人工审核申请提交后有哪些状态？


| 状态      | 含义               |
| ------- | ---------------- |
| **待审批** | 已提交，等待组织管理员处理    |
| **审核中** | 管理员已通过，专家正在鉴定    |
| **已完成** | 专家已提交结论，可查看汇总    |
| **已拒绝** | 管理员拒绝申请（须填写拒绝原因） |


可在 **人工审核申请** 列表或 **检测历史** 详情中查看进度。

### Q23. 如何查看人工审核最终结果？

当申请状态为 **已完成** 且专家已提交结论后：

1. 在 **检测历史** 任务详情中点击 **查看人工审核结果**；或
2. 在 **人工审核申请** 列表进入汇总页（`/manual-review-result`）。

汇总页展示管理员审批意见与专家鉴定结论。

### Q24. 申请被拒绝怎么办？

管理员拒绝时会 **必填拒绝原因**。请根据原因修改材料或补充说明后，对 **新的或符合条件的检测任务** 重新发起申请。

## 六、人工审核（专家）

### Q25. 专家登录后在哪里处理任务？

侧栏 **人工审核**（`/review`）为任务池。仅当组织管理员 **通过** 对应申请后，任务才会出现在池中，显示 **进入审核** 按钮。

### Q26. 专家任务池为空怎么办？

请依次确认：

1. 是否有编辑已发起申请且管理员 **已通过**（非「待审批」或「已拒绝」）。
2. 当前登录的是 **专家账号**（登录页选「专家」）。
3. 审批使用的是 **组织管理员** 账号（非软件管理员只读账号）。
4. 组织内是否存在可用的专家（审稿人）账号。

### Q27. 专家审核时需要填写哪些内容？

进入 **人工审核详情** 后，按任务类型操作：

- **图像类**：查看原图与 AI 参考结果，逐图填写鉴定结论并提交。
- **论文 / Review 类**：查看文本片段、AI 摘要与参考信息，填写鉴定结论并提交。

提交成功后，任务状态变为 **已完成**，编辑端可查看汇总。

## 七、管理端

### Q28. 组织管理员和软件管理员有什么区别？


| 能力            | 组织管理员 | 软件管理员     |
| ------------- | ----- | --------- |
| 人工审核审批（通过/拒绝） | ✅     | ❌（仅可查看队列） |
| 本组织用户与资源管理    | ✅     | 视配置而定     |
| 跨组织检测任务监控     | 本组织范围 | ✅         |
| 模型配置、举报处理     | 视菜单权限 | ✅         |
| 系统日志          | ✅     | ✅         |


人工审核闭环演示与日常审批，请使用 **组织管理员** 账号。

### Q29. 管理员如何审批人工审核申请？

1. 登录管理端，侧栏进入 **人工审核审批**（`/reviews`）。
2. 在列表中找到待处理申请，点击 **详情**。
3. 选择 **通过** 或 **拒绝**：
  - **通过**：任务进入专家任务池，专家可见并处理。
  - **拒绝**：必须填写 **拒绝原因**，否则无法提交。

### Q30. 在哪里查看用户操作与检测失败记录？

侧栏 **系统日志**（`/logs`）包含两个标签：


| 标签         | 内容                              |
| ---------- | ------------------------------- |
| **用户操作留痕** | 登录、上传、检测提交、人工审核申请、专家提交等操作记录     |
| **检测任务记录** | 图像 / 论文 / Review 检测任务及 **失败原因** |


可按类型、时间筛选，支持导出（若界面提供）。

### Q31. 检测任务监控与资源管理在哪里？

- **检测任务**：侧栏 **检测与资源** → **检测任务** 标签，可查看各用户任务状态并筛选。
- **学术资源 / 文件**：同一页面切换至 **学术资源** 标签，管理已上传文件。

## 八、权限、配额与组织

### Q32. 提示「无权访问」或操作被拒绝怎么办？

常见情况：

1. **跨账号访问他人任务**：只能查看和操作 **本人** 发起的检测与申请。
2. **角色权限不符**：如专家无法上传检测、编辑无法进入专家任务池。
3. **账号未绑定组织**：部分上传与检测功能要求用户已加入组织，请联系管理员在后台关联组织。
4. **检测次数或配额不足**：见 Q33。

### Q33. 检测次数有限制吗？

平台按 **用户等级** 与 **组织配额** 控制检测次数（如免费次数、充值次数等，以管理端组织档案显示为准）。若提示配额不足，请联系组织管理员 **充值** 或升级等级。

### Q34. 如何修改个人资料或默认检测模式？

侧栏 **个人主页**（`/profile`）可修改资料、密码；发布者还可设置 **默认检测模式**（标准 / 精准），新建任务时将优先使用该设置。

## 九、通知与社区反馈

### Q35. 如何接收系统通知？

点击用户端顶栏 **铃铛图标** 打开 **通知中心**，可查看广播、审核进度等消息。  
若实时推送不可用，系统会通过 **定时刷新** 更新未读通知，不影响主要业务流程。

### Q36. 社区反馈是做什么用的？

侧栏 **社区反馈**（`/community-feedback`）用于提交对平台的建议、问题反馈或举报相关内容。  
**通知中心** 与社区反馈为不同入口：通知用于接收系统消息，社区反馈用于主动提交意见。

## 十、其他常见问题

### Q37. 访问旧书签或旧链接跳转到别的页面？

系统已对部分历史路径做 **自动重定向**，例如：

- `/detect/paper`、`/detect/review` → 跳转至 **学术检测**（`/upload`）
- 管理端 `/paper-logs`、`/review-logs` → 合并至 **系统日志**（`/logs`）

请使用侧栏当前菜单访问功能，避免依赖已废弃的书签。

### Q38. 平台部署地址是什么？

以管理员提供的地址为准，常见环境：


| 环境     | 用户端                                     | 管理端                     |
| ------ | --------------------------------------- | ----------------------- |
| 课程部署环境 | 见组织通知（如 `http://114.116.204.131:81/` 等） | 同上，管理端入口                |
| 本地联调   | `http://127.0.0.1:3000`                 | `http://127.0.0.1:3001` |


### Q39. 哪些功能目前不在平台承诺范围内？

以下能力 **未纳入当前版本正式验收**，使用时请知悉：

- 大规模 **性能压测**（如固定 3 秒上传、100 并发等指标）
- 依赖 WebSocket 的 **实时推送**（未配置时自动降级为轮询）
- **深度跨模态深度学习融合**（当前为规则层综合报告）
- **DOCX 论文直检**（须转 PDF）
- 单份 PDF 内嵌图片 **一键批量进图像检测**（可用 ZIP/RAR 抽图替代）

不影响主路径：登录 → 检测 → 历史 → 综合报告 → 人工审核闭环。

### Q40. 遇到无法自行解决的问题怎么办？

1. 记录 **操作步骤、页面截图、任务 ID、失败提示**。
2. 在 **社区反馈** 提交问题，或联系本组织管理员。
3. 管理员可在 **系统日志 → 检测任务记录** 中协助排查失败原因。

## 附录：人工审核闭环速查

```
编辑：检测完成 → 发起人工审核申请
        ↓
组织管理员：人工审核审批 → 通过 / 拒绝（拒绝须填原因）
        ↓
专家：人工审核任务池 → 进入审核 → 提交结论
        ↓
编辑：查看人工审核汇总结果
```

**闭环注意**：审批须用 **组织管理员**；专家与编辑请 **分账号、分浏览器或先退出再登录** 操作。
